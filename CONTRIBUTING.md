## Contributing

Thanks for your interest in contributing!

### Development setup

```bash
pnpm i || npm i || yarn
npm run build
```

Run locally with the MCP Inspector:

```bash
npx @modelcontextprotocol/inspector@latest node $(pwd)/dist/index.js
```

### Pull requests
- Keep changes focused and incremental.
- Prefer simple solutions and avoid introducing new tech unless necessary.
- Match existing style and keep files reasonably small (<300 LOC when practical).
- Add or update docs (README) when behavior/config changes.

### Reporting issues
Please include:
- What you did, what happened, what you expected
- Versions (Node, OS) and any relevant logs/output
